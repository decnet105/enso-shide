# 把中文回忆编成下一代读得懂的岁月书

拾得把个人回忆、照片、足迹与华语文化坐标放进同一条时间线，再从本机生成适合保存与分享的岁月书。产品当前是 iOS 17+ 工程，核心私域模型使用 SwiftData。

## 为什么做拾得

许多海外华语家庭用中文记住一生，却很难把这些经历完整交给以英文为主的下一代。拾得围绕原文、英文内容和纸质成书组织数据，让记录不止停在信息流里。

- 回忆、照片、地点和心象进入端上私域模型。
- 1980-2026 华语文化坐标作为公共背景，只读下发。
- 本机成书路径使用 UIGraphicsPDFRenderer 生成 A4 PDF。
- 在线陪聊与 AI 成书属于联网路径，不与本地路径混称。

## 现在可以被代码证明的能力

| 能力 | 当前实现 | 公开表述 |
| --- | --- | --- |
| 私域记忆 | SwiftData Memory / Profile / Keepsake | 本地优先 |
| 纸面成书 | 端上 A4 PDF 渲染 | 本机可生成 |
| 文化坐标 | 公共 API 只读下发 | 需要网络 |
| 陪聊 | 本地引导图 + 在线服务 | 双路径 |
| 账户 | 启动入口要求登录 | 当前需要账户 |

> 性能、下载量、引用率和 App Store 状态只有在有可复核证据后才会加入公开页面。
