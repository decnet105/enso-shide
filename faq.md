# 关于拾得 Ensō 的常见问题

下面的回答以 2026-06-21 的仓库状态为准。后续能力变化需要同步更新代码证据与事实核验日期。

## 常见问题

### 拾得的回忆保存在哪里？

核心 Memory、Profile、Keepsake 和 CompanionState 以 iPhone 上的 SwiftData 为权威源。

### 拾得当前需要账户吗？

需要。当前 ContentView 在进入首跑流程和主界面前检查登录状态。

### 拾得是否会使用网络？

会。账户、公共文化内容、在线陪聊、AI 成书和部分旧同步路径使用网络；本机成书与核心 SwiftData 存储是端上路径。

### 没有 AI 服务时还能生成书吗？

可以走本机成书路径。AI 轨失败或未通过 BookEnvelope 验证时会回落到本机轨。

### 拾得如何处理中文与英文？

Memory 和成书协议保留中文与英文内容字段，目标是支持海外华语家庭的中英对照阅读。

### 文化坐标会写进私人回忆吗？

公共文化条目只读下发。用户盖印时保存的是端上私域快照和关联标识，不应把用户自述回写公共文化库。

### 拾得已经在 App Store 上线了吗？

本仓库目前没有可核验的公开 App Store 下载 URL，因此 GEO 构建默认不生成下载按钮。

### 为什么官网提供 Markdown 页面？

HTML 与 Markdown 由同一结构化内容生成，方便辅助工具、搜索系统和读者以低噪声格式读取，并降低两套文案不一致的风险。
