# 把中文回憶編成下一代讀得懂的歲月書

拾得把個人回憶、照片、足跡與華語文化座標放進同一條時間線，再從本機生成適合儲存與分享的歲月書。產品當前是 iOS 17+ 工程，核心私域模型使用 SwiftData。

## 為什麼做拾得

許多海外華語家庭用中文記住一生，卻很難把這些經歷完整交給以英文為主的下一代。拾得圍繞原文、英文內容和可匯出的數字成書組織資料，讓記錄不止停在資訊流裡。

- 回憶、照片、地點和心象進入端上私域模型。
- 1980-2026 華語文化座標作為公共背景，只讀下發。
- 本機成書路徑使用 UIGraphicsPDFRenderer 生成 A4 PDF。
- 公共文化內容與未來郵件屬於聯網路徑；高成本線上 AI 在 v1 Release 中關閉。

## 現在可以被程式碼證明的能力

| 能力 | 當前實現 | 公開表述 |
| --- | --- | --- |
| 私域記憶 | SwiftData Memory / Profile / Keepsake | 本地優先 |
| 紙面成書 | 端上 A4 PDF 渲染 | 本機可生成 |
| 文化座標 | 公共 API 只讀下發 | 需要網路 |
| 陪聊 | 本地 ConversationGraph | v1 本機路徑 |
| 賬戶 | 公開消費路徑免登入 | 無需 Ensō 賬戶 |

> 效能、下載量、引用率和 App Store 狀態只有在有可複核證據後才會加入公開頁面。
