# 海外家庭怎樣把中文人生故事交給下一代

跨代傳承的難點通常不是缺少照片，而是缺少人物、時間、地點和時代背景。中文原文應被保留，英文內容是閱讀橋樑，不應替代原文。

## 四類需要一起儲存的資訊

| 資訊 | 作用 | 建議 |
| --- | --- | --- |
| 中文原文 | 保留聲音與語氣 | 作為權威文本 |
| 英文內容 | 幫助下一代閱讀 | 與原文並列 |
| 時間地點 | 建立人生座標 | 結構化儲存 |
| 時代背景 | 解釋共同記憶 | 引用可核實公共來源 |

## 為什麼紙面輸出仍然重要

數字書提供穩定順序、章節和共同閱讀場景。持續記錄與可分享 PDF 應來自同一份結構化記憶，而不是分別維護。實體書不屬於拾得 v1 範圍。
