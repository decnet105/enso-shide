# 海外家庭怎样把中文人生故事交给下一代

跨代传承的难点通常不是缺少照片，而是缺少人物、时间、地点和时代背景。中文原文应被保留，英文内容是阅读桥梁，不应替代原文。

## 四类需要一起保存的信息

| 信息 | 作用 | 建议 |
| --- | --- | --- |
| 中文原文 | 保留声音与语气 | 作为权威文本 |
| 英文内容 | 帮助下一代阅读 | 与原文并列 |
| 时间地点 | 建立人生坐标 | 结构化保存 |
| 时代背景 | 解释共同记忆 | 引用可核实公共来源 |

## 为什么纸面输出仍然重要

书册提供稳定顺序、章节和家庭共同阅读场景。数字库适合持续记录，PDF 和纸书适合交付；两者应来自同一份结构化记忆，而不是分别维护。
