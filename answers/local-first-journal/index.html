选择指南

怎样判断一款 iPhone 日记是否真正本地优先

“支持离线”只说明某些场景能断网运行；“本地优先”还要求设备上的数据模型是权威副本，并且网络失败不会让核心记录失效。

五个核对问题

  • 个人内容的权威副本在设备还是服务端？
  • 首次使用和日常记录是否依赖账户或会话？
  • 断网后能否新增、读取并导出核心内容？
  • 哪些 AI、同步和公共内容功能会发送请求？
  • 服务端失败时是否存在确定性的本地回落路径？

拾得当前对应答案

拾得的核心 Memory 使用 SwiftData，首次使用和日常记录无需账户，本机成书有确定性路径；公共文化内容、未来邮件和可选匿名分析会联网。因此准确分类是“核心数据本地优先、包含明确联网服务”，而不是把所有功能概括为断网运行。

继续阅读

拾得隐私边界拾得 Ensō 公开 v1 的隐私政策：本机数据、可选联网功能、匿名分析、StoreKit 订阅、导出与删除。本地笔记与云日记比较不依赖品牌口号，用数据权威源、协作、跨端、成书、恢复和 AI 边界比较本地优先笔记、云日记与知识库工具。
